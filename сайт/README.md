# Каталог порід котів

Це веб-додаток, який демонструє популярні породи котів з описом, характеристиками, фото та фільтрацією.

## Особливості

- Показ популярних порід котів з фото, описом та характеристиками.
- Пошук породи за назвою, коротким описом або характеристиками.
- Фільтр за типом шерсті та гіпоалергенністю:
  - Довгошерсті
  - Короткошерсті
  - Гіпоалергенні / майже
- Модальне вікно з детальною інформацією про породу.
- Адаптивний дизайн для мобільних та десктопних пристроїв.

## Технології

- HTML5, CSS3, JavaScript (Vanilla JS)
- CSS Grid та Flexbox для адаптивного розташування
- Локальні зображення порід (без завантаження користувачем)
